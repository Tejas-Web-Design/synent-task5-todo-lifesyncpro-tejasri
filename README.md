# 🚀 LifeSync Pro

A modern productivity and task management web application designed to help users organize tasks, track progress, maintain daily streaks, and improve productivity through a clean and responsive interface.

---

## 📌 Overview

LifeSync Pro is a personal productivity dashboard that enables users to manage their daily tasks, monitor completion statistics, track streaks, customize their profiles, and personalize their experience with theme settings.

The project is built using pure HTML, CSS, and JavaScript while utilizing Local Storage for data persistence.
